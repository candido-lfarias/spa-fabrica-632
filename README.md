## Visão Geral

Este projeto é uma aplicação **SPA (Single Page Application)** desenvolvida para servir como painel visual e interativo da *Fábrica 632*.  
O objetivo principal é disponibilizar uma interface única, rápida e dinâmica, que permita navegação entre páginas sem recarregar o navegador, tornando o uso mais eficiente.

### Finalidade do sistema

- Visualização e controle de informações internas da fábrica
- Organização de módulos e páginas dentro de um ambiente unificado
- Base para expansão do frontend da Fábrica com React
- Estrutura para futuras integrações com API/back-end

O diretório principal de desenvolvimento é `front-fabrica`, onde fica o código do frontend em React.
